# Project Philosophy

Structora Core is read-only developer infrastructure for structure intelligence and workflow discovery.

It is:

- read-only
- observational
- workflow intelligence
- rendered structure analysis
- developer infrastructure
- schema-oriented
- public-safe

It is not:

- automation software
- browser botting
- bypass tooling
- credential tooling
- workflow execution software
- scraper malware
- payment or checkout execution tooling

The project should help developers understand interface structure without performing user actions or mutating external systems.
